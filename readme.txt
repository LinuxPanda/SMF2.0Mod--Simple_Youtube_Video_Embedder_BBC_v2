[color=purple][size=12pt][b]Features[/b][/size][/color]
[list]
[li]This mod supports both 'youtube.com' & 'youtu.be' links and also http/https links.[/li]
[li]This mod parses youtube links automatically and displays them as videos.
[list]
[li]Youtube videos[/li]
[li]Youtube videos with time identifier[/li]
[li]Youtube playlist[/li]
[/list][/li]
[li]Since youtube video links are automatically parsed & displayed, there is no need to use the 'youtube' tags. At the same time, youtube links & video id's wrapped within 'youtube' tags also work.[/li]
[li]In addition to these features, this mod supports the following format also which was used by some old mods.
[list]
[li][youtube]yVpbFMhOAwE[/youtube][/li]
[li][yt]yVpbFMhOAwE[/yt][/li]
[/list]
[/li]
[li]It works in all browsers (with/without flash). If flash player is not available, then this mod displays the video using HTML5.[/li]
[/list]

[color=purple][size=12pt][b]Who should use this Mod[/b][/size][/color]
If you are one of the people who use only youtube videos & playlist in your forum then this is the mod that you're looking for.


[color=purple][size=12pt][b]How-To[/b][/size][/color]
Youtube video links are parsed & displayed automatically. Just post the youtube links as plain text.

Youtube links wrapped within [nobbc][url][/nobbc] tags won't be parsed as videos. If you wrap a link within [nobbc][url][/nobbc] tags, then it means you are forcing youtube links to be displayed as url.

[color=green][b]Correct:[/b][/color] Hi, checkout this youtube video, [nobbc]http://www.youtube.com/watch?v=yVpbFMhOAwE[/nobbc]

[color=red][b]Incorrect:[/b][/color] Hi, checkout this youtube video, [nobbc][url]http://www.youtube.com/watch?v=yVpbFMhOAwE[/url][/nobbc]
